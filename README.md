# win10-spotlight
获取win10聚焦壁纸

###### 1. 首先机器上要有node环境

[点我去下载](https://nodejs.org/zh-cn/)

###### 2.安装好了后将项目down下来
```
git clone git@github.com:guobing1993/win10-spotlight.git
```

###### 3.进入项目目录
```
cd win10-spotlight
```
###### 4.安装项目依赖
使用npm
```
npm install
```
或者yarn
```
yarn install
```

###### 5.运行项目
```
node index
```



运行后在图片目录下会有spotlight目录
该目录下是win10的聚焦壁纸，包括手机和pc的。

##### 5.另外还可以点击 run.vbs 来运行
